Dynamics.JS

by Bobur Zakhidov

Introduction

Topic: Dynamics.JS is a JavaScript library that creates physics-based animations. Dynamics.js allows you to animate CSS and SVG properties of any DOM elements. And in addition to that, it also allows you to animate any JavaScript objects.

History: It was developed by Michael Villar in mid 2015 and is hosted as open source code on github. Soem code from Sylvester.js (vector and matrix parts) has been used while developing dynamics.js. It supports the following browsers: Safari (7++), Firefox (35++), Chrome (34++),and IE (10++). Latest version is 1.1.5 and was last updated on August 26, 2016.

Importance and Relavance: I find this topic to be closely related to what we are being taught in CIS 3344 and that is manupulation of DOM elements using JavaScript. Dynamics.js manipulates DOM elements and adds physics animation to them. And anyone with basic knowledge of JavaScript will be able to use it in their own project.

Set Up

Required tools: HTML document editor (optional), Visual Studio, and browser (Chrome 34++ prefered)

Download: There are 3 ways to download Dynamics.JS:

  1. Using npm package manager

    2. Type the following command: npm install dynamics.js

  2. Using bower package manager

    3. Type the following command: bower install dynamics.js

  3. Downloading the required files from github page

    4. Link to download: https://github.com/michaelvillar/dynamics.js/releases

Setup: In order to start using dynamics.js, follow these steps:

  1. Extract the downloaded file
  2. Navigate the file and find dynamics.js file
  3. Copy the dynamics.js file into your project folder

Finish: Now you are ready to use the dynamics.js
