Lab 1

Overall, my experience with front-end is very limited. I had previous experience in using HTML, CSS, and JavaScript when I took Server Side Scripting course. Although I did not use any advanced features because the pages I created were very simple. Going forward, the only framework I had exposure to was Bootsrap which I utilized in my Capstone 1 course.

The advanced CSS techniques that I used for this lab were page layouts and box models. I heavily used floating techniques to make the content appear the way I want. I also used box model to get the right width of the element. Lastly, I incorporated responsive design to one the pags, where the content will display differently when screen is re-sized.

I liked using page layout and box modeling techniques as they helped a lot with making the website look fluid and responsive. I also liked using navigation menus to navigate through different pages. With that being said, the only thing I disliked doing was finding the right width of elements, because it involves adding padding, border, margin and content size, which sometimes was confusing to get.
